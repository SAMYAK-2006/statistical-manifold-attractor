# Statistical Manifold Attractor (SMA) Framework

**Author:** Samyak Jain  
**Affiliation:** NIT Rourkela  
**Date:** March 2026 (Preprint)

## Overview

This repository contains the preprint for the **Statistical Manifold Attractor (SMA) Framework**, a geometric approach to regime dynamics in non-stationary time series.

Instead of modeling regimes as discrete labels, the SMA framework represents the statistical state of a process as a trajectory on a **statistical parameter manifold**.

Key quantities:

- Statistical feature vector θ(t)
- Velocity and speed of regime change
- Attractor distance D(t)
- Displacement persistence P(t, W)
- Data-trust score α(t)

## Key Contributions

1. **Speed as a regime transition detector**
2. **Universal mean reversion in statistical parameter space**
3. **Statistical attractor dynamics**
4. **Data-trust score for decision systems**

Validated across:

- Finance (S&P 500)
- Solar physics (sunspots)
- Epidemiology (COVID-19)
- Climate science (global temperature)
